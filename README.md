# Exercism

This repository contains my solutions for [Exercism](https://exercism.org) questions.

Each problem only contains the solution file for simplicity.
Tests, build files, READMEs and other miscellaneous files are not here.
Therefore, to run these solutions, it is recommended to pull the problem files from
the [Exercism CLI Tool](https://exercism.org/docs/using/solving-exercises/working-locally)
and replace the template file with the solution file.
